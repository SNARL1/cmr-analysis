54188 - Analyze capture-mark-recapture data using mrmr

Roland Knapp

This notebook analyzes capture-mark-recapture data using the mrmr package. The code creates the required survey, capture, and translocation data frames from saved CSV files (created by 54188_createdata_mrmr.Rmd), checks the datasets, fits the model, evaluates potential non-convergence of the MCMC algorithm, and visualizes model results.

Before using the code in this file, replace all instances of “xxxxx” with the 5-digit site_id of interest.
