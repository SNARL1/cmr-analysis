Identify which coefficient corresponds to which treatment

# colnames(model.matrix(~ treatment, data = data_22008$survival_covariate_df))
  • Column names are “Intercept” and “treatmentexposed”. That indicates the following:
    • beta_phi[1] = intercept = survival for the baseline group (unexposed)
    • beta_phi[2] = effect of treatment = exposed relative to the baseline
  • Model formula incidates the same, without having to read from data_22008.

Compute survival probabilities

# For convenience
beta1 <- beta_phi_post$`beta_phi[1]`   # Intercept (unexposed reference level)
beta2 <- beta_phi_post$`beta_phi[2]`   # Effect of exposed

# Survival probabilities
phi_unexposed <- plogis(beta1)
phi_exposed <- plogis(beta1 + beta2)
  • beta_phi parameters on logit scale (= log-odds scale), plogis converts log-odds into survival probabilities
  • beta_phi[1] = logit(survival of unexposed, the baseline)
  • beta_phi[2] = log-odds difference for “exposed” relative to unexposed: beta_phi[1] + beta_phi[2]

Put results in tidy dataframe

surv_probs <- tibble(
  exposed = phi_exposed,
  unexposed = phi_unexposed
) %>%
  pivot_longer(everything(), names_to = "treatment", values_to = "phi")

-0.18595915 -0.05200765  0.04392502
  • On average, across the posterior, exposed frogs have ~5.7% lower survival probability than exposed frogs.
  • Interpretation of Bayesian credible interval: Given the model and data, there is a 95% probability that the true difference in survival (exposed – unexposed) lies between –18.6% and +4.4%.
  • The most likely difference is a small negative effect (exposed survive slightly worse). However, the credible interval overlaps zero, so the data do not strongly support either a real decrease in survival from exposure, or a real increase.
